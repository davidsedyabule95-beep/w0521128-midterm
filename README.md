# Tank Game UML and OOP Demonstration

## Project Overview

This project demonstrates basic Object-Oriented Programming (OOP) concepts using a simple tank game concept. The project includes UML diagrams and Python code that represent the structure and behavior of the game objects.

## UML Diagrams

Two UML diagrams were created:

**Class Diagram**

* Shows the main classes used in the game.
* Classes include GameLevel, Maze, and Tank.
* Each class has attributes, methods, and relationships with other classes.

**Sequence Diagram**

* Shows the order of actions when the game starts.
* Steps include creating the player, loading the maze, spawning enemies, and initializing the score system.

## Python Implementation

The Python code demonstrates three key OOP principles:

**Encapsulation**

* Class attributes such as walls in the Maze class are protected.

**Inheritance**

* Player and Enemy classes inherit from the Character base class.

**Polymorphism**

* The move() method behaves differently for Player and Enemy.
